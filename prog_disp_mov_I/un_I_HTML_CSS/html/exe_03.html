Server Side / Client Side

Server Side: É a linguagem de programação executada e processada no servidor, antes de enviar o resultado para o cliente. O código Server Side é responsável por gerar e entregar as páginas da web prontas para o navegador do cliente. Exemplos de linguagens Server Side incluem:

  • Python

  • Java

  • NodeJS

Client Side: É a linguagem de programação executada e processada no navegador do cliente. O código Client Side é responsável por manipular a interface do usuário e realizar operações no próprio navegador, sem a necessidade de fazer requisições adicionais ao servidor. Exemplos de linguagens Client Side incluem:

  • JavaScript

  • HTML

  • CSS
